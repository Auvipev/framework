A little MVC framework for undersand and learn PHP programming

Features:
 
   * Model View Controller
   * Session Managment and Login
   * Routing
   * Dependency Injections
   * Storage Manager for data base and cache
   * PSR4 Autoloader

Require:

   * PHP 7 or higher
